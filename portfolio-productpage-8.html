Product Detail Page

구매 전환을 고려한 모바일 스크롤 설계

Overview

모바일 커머스 환경에 최적화된 제품 상세 페이지(Detail Page) 디자인입니다. 단순한 정보 나열을 넘어, 소비자의 구매 여정(Customer Journey)을 고려하여 신뢰도 형성 → USP(강점) 소구 → 구매 장벽 해소의 단계적 흐름으로 기획했습니다. 상단 훅(Hook) 요소로 브랜드를 강조하고, 중단에서 시각화된 데이터로 강점을 구체화하며, 하단의 Q&A 배치로 이탈률을 최소화하는 UX 구조를 구축했습니다.

Design Strategy

  • 모바일 UX 스크롤링 : 모바일 디바이스의 세로 스크롤 호흡에 맞춘 레이아웃 및 폰트 스케일 적용
  • 구매 전환 구조 설계 : 사용자의 설득 과정을 논리적인 섹션 배치로 유도하는 퍼널(Funnel) 기획
  • 정보의 시각적 청킹(Chunking) : 긴 텍스트를 배제하고 아이콘과 인포그래픽 중심의 블록 단위 시각화
  • 전략적 CTA 배치 : 스크롤이 머무는 핵심 구간마다 행동 유도 버튼(CTA)을 배치하여 전환율(CVR) 극대화

Retrospective

  • 상단 배너 및 메인 카피에 대한 A/B 테스트 기반 성과 측정 구조 고민
  • 소비자 구매 후기(Social Proof) 영역의 시각적 강조 및 UI 개선안 고도화
  • 디바이스(iOS/Android)별 안전 영역(Safe Area)을 고려한 CTA 위치 최적화 테스트
  • 데이터(스크롤 뎁스, 이탈률)에 기반한 상세 페이지 지속 개선 프로세스 훈련 필요
